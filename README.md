# Test FE - Class Website Frontend

React + Vite frontend connected to backend authentication APIs.

## Backend APIs used

- `POST /api/auth/register`
- `POST /api/auth/login`

## Environment

Copy `.env.example` to `.env` and set:

- `VITE_API_BASE_URL` (example: `http://localhost:3000`)

## Run

1. `npm i`
2. `npm run dev`

## Build

- `npm run build`
